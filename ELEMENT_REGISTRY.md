# Element Registry

This document defines the supported HTML elements in **HTMLForge v1.1** and their classification.

The registry is **authoritative**: if an element is not listed here, it is not supported.

## Notes

- Deprecated elements are **intentionally excluded**
- SVG and MathML are **out of scope** for v1.x
- Custom elements are **not supported**

This registry is frozen for v1.1.x.
